## Overview
Blood Bridge is a real-time platform to help donors, blood banks, and hospitals coordinate effectively during emergencies. The `rednova-ui` repository contains the frontend UI and related documentation for the Blood Bridge project.

## Tech stack (high level)
- Frontend: React + Tailwind CSS
- Backend: Node.js (Express)
- Database: MongoDB
- Realtime & Notifications: WebSocket / Push / Third-party services
- Location: Geo APIs for mapping & routing
- CI: GitHub Actions
